# LivroDeErros - Base de Conhecimento Portátil

Sistema desenvolvido para catalogar erros técnicos, causas e soluções de forma rápida e eficiente. Ideal para desenvolvedores que precisam de uma base de consulta "pocket" (em pendrive).

## Tecnologias
- **Backend:** Python com FastAPI
- **Banco de Dados:** SQLite (SQLAlchemy)
- **Frontend:** HTML5 & Tailwind CSS
- **Portabilidade:** PyInstaller (Geração de .exe)

## Diferenciais
- **Zero Instalação:** Roda direto de um executável.
- **Banco de Dados Local:** Arquivo `.db` integrado que viaja com o app.
- **Busca por Código:** Filtro rápido por códigos de erro.
